RAF-ticker

A simple ticker with requestAnimationFrame.

Getting Started

Install

 
$ npm install https://github.com/MasatoMakino/raf-ticker.git -D

How to use

 
import { RAFTicker, RAFTickerEventType } from "raf-ticker";

RAFTicker.on(RAFTickerEventType.tick, (e) => {
  console.log(e);
});
